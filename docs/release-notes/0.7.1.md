# PoshMcp v0.7.1 Release Notes

**Released**: 2026-04-19

## What's New

### Program.cs Refactoring

The monolithic `Program.cs` has been refactored to improve code maintainability and testability. Five utility classes have been extracted:

- **`LoggingHelpers`** — Centralized logging configuration setup
- **`DockerRunner`** — Docker build and execution operations
- **`SettingsResolver`** — Configuration resolution from CLI, environment, and appsettings
- **`ConfigurationFileManager`** — File-based configuration read/write operations
- **`ConfigurationLoader`** — Loads and validates configuration from multiple sources

This refactoring improves code clarity and makes it easier to add new configuration features in future releases.

## Bug Fixes & Stability

- **Fixed Docker build command** — Added missing build context path to `docker buildx build` command, resolving build failures when using Docker Buildx

## Breaking Changes

None. This is a maintenance and refactoring release.

## Upgrade Notes

### For Existing Deployments

- No configuration changes required
- No breaking changes to API or CLI behavior
- Safe to upgrade immediately

## Platform Support

- **Windows**: .NET 10 runtime
- **Linux/macOS**: .NET 10 runtime
- **Docker**: Included in `poshmcp:latest` image

## Testing

This release includes:
- Regression tests for Docker build operations
- Unit tests for refactored utility classes
- Integration tests for configuration resolution across all sources
